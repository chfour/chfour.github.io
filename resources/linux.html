linux/unix system management

user management

  • adding users: # useradd -m name
  • removing users: # userdel name (use -r to remove home directory)
  • change password: $ passwd [user]
  • sudo as user: $ sudo -i -u user command
  • shell as user: # su user (optionally add -l for a login prompt)

file management

  • copy file: $ cp from to
  • move/rename file: $ mv from to
  • symlink file: $ ln -s from to

file permissions

  • set file owner: $ chown user[:group]
  • set file permissions: $ chmod ugo file
    u - permissions for owner
    g - permissions for owner group
    o - permissions for others
    (see below)
  • get file permissions: $ stat -c %a file

file permission integers

  • 1: executable (x)
  • 2: write (w)
  • 4: read (r)
  • add to combine (eg. 7 = RWX, 6 = RW etc.)
